# Safety

Use A11yway only on public pages or pages you have permission to review.

A11yway does not:

- log in to private systems
- bypass authentication
- submit arbitrary public forms
- make payments
- exploit vulnerabilities
- scrape private data
- certify legal compliance

Browser and workflow checks are built for accessibility evidence. Passive security checks are opt-in and separate from accessibility findings. They inspect provided markup, URLs, and normal response metadata only.
