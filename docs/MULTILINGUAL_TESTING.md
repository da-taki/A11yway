# Multilingual And Bidirectional Testing

`--language-audit` expands language review beyond the original Indic checks.

It conservatively detects sustained Devanagari, Gurmukhi, Bengali, Tamil, Telugu, Arabic/Urdu, and Hebrew script evidence, missing passage language tags, RTL direction concerns, and non-English accessible names lacking local language metadata.

Proper names, short phrases, code snippets, and isolated words require manual judgment.
